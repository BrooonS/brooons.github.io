description Описание

С помощью этого плагина вы можете легко поставить таймер на ваш сайт, он работает в обоих направлениях. С помощью конфигурации вы можете изменить теги как букв и цифр, вы также можете изменить текстовый вывод рядом с цифрами.

compare_arrows Два направления

Вы указали дату, которая уже вот-вот закончится? TimezZ продолжит сообщать, сколько времени прошло с указанной даты.

info_outline Две версии

Можно использовать две версии, одна версия - это версия для современных браузеров со стандартами ES2017 и вторая версия для старых браузеров со стандартами ES2015.

build Использование

Если вам нужна поддержка более старых браузеров, таких как IE9, используйте jquery.timezz-es2015.js

  • Подключение

    <script src="/js/jquery.timezz.min.js"></script>

  • HTML

    <div id="timer"></div>

  • Инициализация

    $('#timer').timezz();

list Конфигурация

$('#timer').timezz({
'date' : 'January 1, 2040 00:00:00',
'days' : 'd',
'hours' : 'h',
'minutes' : 'm',
'seconds' : 's',
'tagNumber' : 'span',
'tagLetter' : 'i',
'stop' : false
});
Настройка По умолчению Описание Тип
date January 1, 2040 00:00:00 Указанная дата string
days d Как называть дни string
hours h Как называть часы string
minutes m Как называть минуты string
seconds s Как называть секунды string
tagNumber span Тег, в который обернуты цифры string
tagLetter i Тег, в который обернуты буквы string
stop false Включен ли таймер? boolean

settings Пример кастомизации

#timer {
  font-size: 70px;
}
#timer span {
  color: #555;
}
#timer i {
  color: #bbb;
  font-size: 40px;
}
